Milo ClaudeKit
===

A collection of plugins for Claude Code.

![ClaudeKit Introduction](./assets/claudekit-intro.gif)

## Installation

In `claude` run:

```
/plugin marketplace add Michael0520/milo-claudekit
/plugin install <plugin-name>@milo-claudekit
```
